fosserver
==========

fosserver. python 2 and 3 compatible.

Dependency
==========

- python 2 or 3

- tornado

- six

- redis

- pony

Usage
=====

#### Setup environment

```
pip install virtualenvwrapper
./setup_env.sh
source ~/.zshrc  # if use zshrc
workon fosserver
```

#### Run

`python src/main.py`
